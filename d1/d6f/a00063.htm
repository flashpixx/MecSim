Detailed Description

Note
Jason needs on the Agent.create call an instance of AgArch and not AgArchTier, so we need an own class to create an own cycle call
Warning
An AgArch is a linked-list of AgArchs, the agent name can read if an AgArch has got a successor only (Jason code design error)
